# CLAUDE.md

- Ce projet consiste à élaborer un cv simple et interactif avec une page html (index.html).
- Les fichiers css et js doivent être dans des fichiers à part et non pas directement dans le fichier index.html.

- Ce projet doit être responsive, adapté aux mobiles.

Le cv doit comporter les catégories suivantes :
- Un titre
- Un chapô avec un petit texte d'introduction
- La liste des expériences
- Les formations
- Les compétences
- Les certifications
- Les langues

Les différentes catégories à part le chapô et le titre doivent être dans des "accordions".


## Commands
Ne lances pas de pré-rendu, ne teste pas le pré-rendu.
